# Armonia Excursions - 디자인 아이디어

## 선택된 디자인 철학: 미니멀 럭셔리 + 자연 감성

### 디자인 무브먼트
**Contemporary Minimalism with Organic Elements** - 현대적 미니멀리즘과 자연의 유기적 형태를 결합한 프리미엄 여행사 브랜드

### 핵심 원칙
1. **공간의 우아함**: 넉넉한 여백과 명확한 계층 구조로 고급스러움 표현
2. **자연과의 조화**: 그리스 풍경, 고대 유적의 이미지를 통해 목적지의 본질 전달
3. **타이포그래피 대비**: 굵은 세리프 폰트(제목)와 가독성 높은 산세리프(본문) 조합
4. **제약된 색상**: 따뜻한 베이지, 진한 녹색, 화이트로 자연스러운 우아함 구현

### 색상 철학
- **배경**: 따뜻한 베이지/크림색 (자연, 흙, 고대 건축물)
- **주요 텍스트**: 진한 슬레이트 그린 (신뢰, 자연, 안정성)
- **강조색**: 골드/앰버 (프리미엄 감각, 햇빛)
- **CTA 버튼**: 깊은 녹색 (행동 유도, 자연과의 연결)

### 레이아웃 패러다임
- **비대칭 구성**: 이미지와 텍스트의 불규칙한 배치로 동적 느낌
- **카드 기반 섹션**: 각 서비스/투어를 독립적인 카드로 표현
- **이미지 우선**: 시각적 스토리텔링에 중점, 텍스트는 보조적 역할
- **수직 리듬**: 섹션 간 일관된 간격으로 시각적 흐름 유지

### 시그니처 요소
1. **둥근 모서리 카드**: 모든 콘텐츠 박스에 일관된 border-radius 적용
2. **미묘한 그림자**: 깊이감을 주는 소프트 섀도우
3. **이미지 오버레이**: 텍스트 가독성을 위한 그라데이션 오버레이

### 인터랙션 철학
- **부드러운 전환**: 모든 호버 상태에서 부드러운 색상/스케일 변화
- **명확한 피드백**: 버튼 클릭 시 시각적 반응
- **스크롤 애니메이션**: 섹션 진입 시 fade-in 효과

### 애니메이션 가이드
- 페이지 로드: 섹션별 스태거 페이드인 (200ms 간격)
- 호버 상태: 0.3s ease-out 스케일 및 색상 변화
- 버튼: 0.2s ease-in-out 배경색 전환
- 스크롤: Intersection Observer를 활용한 진입 애니메이션

### 타이포그래피 시스템
- **제목**: Playfair Display (세리프, 대담함, 고급스러움)
- **본문**: Lato (산세리프, 가독성, 현대성)
- **계층 구조**:
  - H1: 48px, 700 weight (페이지 제목)
  - H2: 36px, 600 weight (섹션 제목)
  - H3: 24px, 600 weight (카드 제목)
  - Body: 16px, 400 weight (본문)
  - Small: 14px, 400 weight (보조 텍스트)
